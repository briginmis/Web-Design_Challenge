Does latitude impact the weather?

The purpose of this project is to determine whether a city's latitude impacts temperature, humidity, cloudiness and wind speed. We hypothesize that latitudes closer to 0 will have higher temperatures, as it is closer to the sun.


We undertook this analysis by performing the following steps:

  1. Randomly selected over 200 latitude and longitude coordinates.
  2. Used Google Places API to identify the closest city to the selected coordinates.
  3. Used Weather API to obtain the temperature, humidity, cloudiness and wind speed for each city.
  4. Plotted the latitude and temperature, humidity, cloudiness and wind speed on a scatter plot.
  5. Used regression analysis to determine whether there is a correlation between latitude and temperature, humidity, cloudiness and wind speed.
